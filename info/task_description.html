You prefer good old 12-hours time format. But modern world live in 24-hours format and you see it everywhere. Your task is to convert time from 24-h format into 12-h using the next rules:
- the output format should be 'hh:mm a.m.' (for hours before midday) or 'hh:mm p.m.' (for hours after midday)
- if hours is less than 10 - don't write a '0' before it. For example: '9:05 a.m.'
You can find some useful information about 12-hours format.

Input: Time in 24-hours format (as string).

Output: Time in 12-hours format (as string).

Precondition:
